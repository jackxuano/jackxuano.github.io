## What's inside
- Hole maps, shot plans, caddie notes, green reads
- Live weather forecast at page load
- Wind effect per hole
- Blue / White / Red tee toggle
- Password protected

## Using it for your own course
Shot plans are calibrated to my carry distances — swap them for yours.
You'll need your own hole images and scorecard. Everything else is reusable.
